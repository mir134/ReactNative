# 小度鱼（React-Native）
小度鱼一个网络社区系统

## 功能介绍
+ 支持QQ、微博、GitHub、邮箱，注册登陆绑定
+ 支持新评论、新回复的时候发送通知消息
+ 使用DraftJS，实现客户端简单富文本（带上传图片）
+ 支持上传图片到七牛
+ 以及上传头像、裁剪头像、修改昵称、个性签名、WebSocket获取新通知等等
